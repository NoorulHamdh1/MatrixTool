# 🧮 Matrix Operations Tool

A modern web-based Matrix Operations Tool built using **Flask** and **NumPy**. The application allows users to perform common matrix operations through an interactive and user-friendly interface without requiring command-line input.

---

## 🚀 Features

- ➕ Matrix Addition
- ➖ Matrix Subtraction
- ✖️ Matrix Multiplication
- 🔄 Matrix Transpose
- 📐 Matrix Determinant
- ⚠️ Error handling for invalid matrix dimensions and inputs
- 💻 Clean, responsive web interface

---

## 🛠️ Technologies Used

- Python
- Flask
- NumPy
- HTML5
- CSS3

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/NoorulHamdh1/Matrix-Operations-Tool.git
```

### Navigate to the project

```bash
cd Matrix-Operations-Tool
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
