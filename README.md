# Matrix Operations Tool

A web-based Matrix Operations Tool developed using Flask and NumPy. The application enables users to perform common matrix operations through an interactive interface without using the command line.

## Features

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose
- Matrix Determinant
- Error handling for invalid matrices
- Clean and responsive interface

## Technologies Used

- Python
- Flask
- NumPy
- HTML
- CSS
